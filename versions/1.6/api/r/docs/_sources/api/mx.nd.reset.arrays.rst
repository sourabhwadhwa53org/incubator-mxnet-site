

``mx.nd.reset.arrays``
============================================

Description
----------------------

Set to zero multiple arrays.  



Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``data``                               | NDArray-or-Symbol[].                                       |
|                                        |                                                            |
|                                        | Arrays                                                     |
+----------------------------------------+------------------------------------------------------------+
| ``num.arrays``                         | int, required.                                             |
|                                        |                                                            |
|                                        | number of input arrays.                                    |
+----------------------------------------+------------------------------------------------------------+

Value
----------

``out`` The result mx.ndarray
