

``mx.gpu``
====================

Description
----------------------

Create a mxnet GPU context.


Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``dev.id``                             | optional, default=0.                                       |
|                                        |                                                            |
|                                        | The GPU device ID, starts from 0.                          |
+----------------------------------------+------------------------------------------------------------+

Value
----------

``The`` GPU context.
