

``mx.serialize``
================================

Description
----------------------

Serialize MXNet model into RData-compatiable format.

Usage
----------

.. code:: r

	mx.serialize(model)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``model``                              | The mxnet model                                            |
+----------------------------------------+------------------------------------------------------------+
