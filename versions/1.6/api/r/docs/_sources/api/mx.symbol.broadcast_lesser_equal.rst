

``mx.symbol.broadcast_lesser_equal``
========================================================================

Description
----------------------

Returns the result of element-wise **lesser than or equal to** (<=) comparison operation with broadcasting.


**Example**::

	 
	 x = [[ 1.,  1.,  1.],
	 [ 1.,  1.,  1.]]
	 
	 y = [[ 0.],
	 [ 1.]]
	 
	 broadcast_lesser_equal(x, y) = [[ 0.,  0.,  0.],
	 [ 1.,  1.,  1.]]
	 
	 
	 

Usage
----------

.. code:: r

	mx.symbol.broadcast_lesser_equal(...)
