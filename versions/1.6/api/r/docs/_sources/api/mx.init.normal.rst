

``mx.init.normal``
====================================

Description
----------------------

Create a initializer that initialize the weight with normal(0, sd)

Usage
----------

.. code:: r

	mx.init.normal(sd)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``sd``                                 | The standard deviation of normal distribution              |
+----------------------------------------+------------------------------------------------------------+
