Detailed Description

The data type the tensor can hold.

Examples

  • float: type_code = 2, bits = 32, lanes=1
  • float4(vectorized 4 float): type_code = 2, bits = 32, lanes=4
  • int8: type_code = 0, bits = 8, lanes=1
