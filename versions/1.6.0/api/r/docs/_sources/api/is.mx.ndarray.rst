

``is.mx.ndarray``
==================================

Description
----------------------

Check if src.array is mx.ndarray

**Example**::

	 mat = mx.nd.array(1:10)
	 is.mx.ndarray(mat)
	 mat2 = 1:10
	 is.mx.ndarray(mat2)
	 
Usage
----------

.. code:: r

	is.mx.ndarray(src.array)


Value
----------

Logical indicator
