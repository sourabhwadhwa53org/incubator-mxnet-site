

``mx.ctx.default``
====================================

Description
----------------------

Set/Get default context for array creation.

Usage
----------

.. code:: r

	mx.ctx.default(new = NULL)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``new``                                | optional takes ``mx.cpu()`` or ``mx.gpu(id)``, new default |
|                                        | ctx.                                                       |
+----------------------------------------+------------------------------------------------------------+

Value
----------

The default context.
