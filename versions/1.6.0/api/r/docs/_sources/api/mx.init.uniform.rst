

``mx.init.uniform``
======================================

Description
----------------------

Create a initializer that initialize the weight with uniform [-scale, scale]

Usage
----------

.. code:: r

	mx.init.uniform(scale)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``scale``                              | The scale of uniform distribution                          |
+----------------------------------------+------------------------------------------------------------+
