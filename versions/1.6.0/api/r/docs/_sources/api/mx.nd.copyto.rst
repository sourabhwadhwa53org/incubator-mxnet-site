

``mx.nd.copyto``
================================

Description
----------------------

Generate an mx.ndarray object on ctx, with data copied from src

Usage
----------

.. code:: r

	mx.nd.copyto(src, ctx)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``src``                                | The source mx.ndarray object.                              |
+----------------------------------------+------------------------------------------------------------+
| ``ctx``                                | The target context.                                        |
+----------------------------------------+------------------------------------------------------------+
