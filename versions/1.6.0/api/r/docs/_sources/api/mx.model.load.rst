

``mx.model.load``
==================================

Description
----------------------

Load model checkpoint from file.

Usage
----------

.. code:: r

	mx.model.load(prefix, iteration)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``prefix``                             | string prefix of the model name                            |
+----------------------------------------+------------------------------------------------------------+
| ``iteration``                          | integer Iteration number of model we would like to load.   |
+----------------------------------------+------------------------------------------------------------+
