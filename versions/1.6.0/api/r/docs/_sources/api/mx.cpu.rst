

``mx.cpu``
====================

Description
----------------------

Create a mxnet CPU context.


Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``dev.id``                             | optional, default=0.                                       |
|                                        |                                                            |
|                                        | The device ID, this is meaningless for CPU, included for   |
|                                        | interface                                                  |
|                                        | compatiblity.                                              |
+----------------------------------------+------------------------------------------------------------+

Value
----------

``The`` CPU context.
