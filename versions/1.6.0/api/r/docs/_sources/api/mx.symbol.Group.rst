

``mx.symbol.Group``
======================================

Description
----------------------

Create a symbol that groups symbols together.

Usage
----------

.. code:: r

	mx.symbol.Group(...)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``kwarg``                              | Variable length of symbols or list of symbol.              |
+----------------------------------------+------------------------------------------------------------+

Value
----------

The result symbol
