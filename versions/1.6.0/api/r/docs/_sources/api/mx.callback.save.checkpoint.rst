

``mx.callback.save.checkpoint``
==============================================================

Description
----------------------

Save checkpoint to files each period iteration.

Usage
----------

.. code:: r

	mx.callback.save.checkpoint(prefix, period = 1)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``prefix``                             | The prefix of the model checkpoint.                        |
+----------------------------------------+------------------------------------------------------------+
