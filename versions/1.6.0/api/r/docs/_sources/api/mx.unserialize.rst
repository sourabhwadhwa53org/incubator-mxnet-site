

``mx.unserialize``
====================================

Description
----------------------

Unserialize MXNet model from Robject.

Usage
----------

.. code:: r

	mx.unserialize(model)

Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``model``                              | The mxnet model loaded from RData files.                   |
+----------------------------------------+------------------------------------------------------------+
