

``mx.symbol.Variable``
============================================

Description
----------------------

Create a symbolic variable with specified name.


Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``name``                               | string.                                                    |
|                                        |                                                            |
|                                        | The name of the result symbol.                             |
+----------------------------------------+------------------------------------------------------------+

Value
----------

The result symbol
