

``mx.nd.logical.not``
==========================================

Description
----------------------

Returns the result of logical NOT (!) function.  

Example:
  logical_not([-2., 0., 1.]) = [0., 1., 0.]


Arguments
------------------

+----------------------------------------+------------------------------------------------------------+
| Argument                               | Description                                                |
+========================================+============================================================+
| ``data``                               | NDArray-or-Symbol.                                         |
|                                        |                                                            |
|                                        | The input array.                                           |
+----------------------------------------+------------------------------------------------------------+

Value
----------

``out`` The result mx.ndarray
